{frontmatter.description}

Lovely Guruji is an eLearning system based website. It uses resources from varoius sources to help you know about a vast range of technical knowledge and skills. 

**The live demo is at [here](https://lovelyguruji.vercel.app/)**

This website is built with the help of the popular react library [Nextjs.](https://nextjs.org/)

## Tech Stack

The tech stack for this project was:
- [React](https://reactjs.org/)
- [Nextjs](https://nextjs.org/)
- [Tailwindcss](https://tailwindcss.com/) 
- [MDX](https://mdxjs.com/)
- [Vercel](https://vercel.com/)
- [Endpoint](https://endpoint.space/)
- [Analytics](https://analytics.google.com/)

The frontend is built with the help of nextjs and mdx. For the styling tailwindcss was used. To collect the form repsonse the help of [endpoint space](https://endpoint.space/) was used. And at the end to host the website vercel was used along with google analytics to measure the viwer stats, etc.
